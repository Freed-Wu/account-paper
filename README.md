account-paper
=============

A backup of my homework.

Dependent
---------

1.  A LaTex distribution. Such as [texlive].
2.  [njustthesis].

Install
-------

``` {.zsh}
git clone git@github.com:Freed-Wu/account-paper
cd account-paper
latexmk -pvc main.tex
```
